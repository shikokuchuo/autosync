# autosync (development version)

* `amsync_project()` browses a project document's file tree from just a server
  URL and project ID, and edits files by path.
* `amsync_edit()` opens a synced text object in your `$EDITOR`, merging your
  edits back into the live document on save while preserving concurrent remote
  edits.
